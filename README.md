# F1 Data Pipeline

An automated data pipeline that collects and analyzes Formula 1 race data to provide insights into driver performance, race strategy, and team dynamics.

## Overview

This project builds a comprehensive database of F1 race session data, focusing on telemetry, timing, and strategic information from Grand Prix weekends. The pipeline enables data-driven analysis of:

- Race performance metrics
- Driver and team comparisons
- Strategic decision patterns
- Circuit-specific characteristics
- Season-long trends

##  Data Model

The pipeline maintains structured data across key F1 domains:

- **Race Events**: Circuit information and session details
- **Driver Data**: Performance statistics and team affiliations
- **Timing Data**: Lap times, sector splits, and speed measurements
- **Strategy Analytics**: Pit stop timing and tyre management
- **Race Results**: Final standings and competitive outcomes
- **Telemetry**: Aggregated car performance data

## Analytics Capabilities

The processed data enables analysis of:

- **Performance Analysis**: Compare driver lap times, sector performance, and speed trap data
- **Race Strategy**: Analyze pit stop timing, tyre compound choices, and fuel management
- **Team Dynamics**: Head-to-head performance metrics and strategic patterns
- **Circuit Insights**: Track-specific performance characteristics and challenges
- **Season Trends**: Year-over-year performance improvements and competitive shifts
